# Hyprland Dotfiles

This repository contains my personal [Hyprland](https://github.com/hyprwm/Hyprland) dotfiles.

## Dependencies

Make sure you have these installed for everything to work properly:

* [`Hyprland`](https://github.com/hyprwm/Hyprland) - WM
* [`Eww`](https://github.com/elkowar/eww) – Bar
* [`Cava`](https://github.com/karlstav/cava) – Audio visualizer
* [`Rofi`](https://github.com/davatorium/rofi) – App Launcher
* [`Fish`](https://github.com/fish-shell/fish-shell) - Shell
* [`Starship`](https://github.com/starship/starship) - Shell Prompt
* [`Kitty`](https://github.com/kovidgoyal/kitty) - Terminal
* [`Fastfetch`](https://github.com/fastfetch-cli/fastfetch) - System Information 
* [`Btop`](https://github.com/aristocratos/btop) – Resource Monitor
* [`playerctl`](https://github.com/altdesktop/playerctl), [`amixer`](https://www.alsa-project.org/) **or** [`pamixer`](https://github.com/cdemoulins/pamixer) – Volume Control
* [`hyprlock`](https://github.com/hyprwm/hyprlock), [`swaylock`](https://github.com/swaywm/swaylock) **or** [`i3lock`](https://github.com/i3/i3lock) – Lockscreen
* [`hyprctl`](https://github.com/hyprwm/Hyprland#hyprctl), [`jq`](https://github.com/stedolan/jq) - JSON parsing
* [`systemctl`](https://www.freedesktop.org/software/systemd/man/systemctl.html) - System Management
* [`date`](https://www.gnu.org/software/coreutils/manual/html_node/date-invocation.html) - Date command
* [`df`](https://www.gnu.org/software/coreutils/manual/html_node/df-invocation.html), [`free`](https://www.gnu.org/software/procps/) - Disk & memory usage
* [`Fonts`](https://www.nerdfonts.com/font-downloads) JetBrainsMono Nerd Font, Iosevka Nerd Font Mono

## Setup / Installation
> [!CAUTION] 
> BACKUP YOUR CURRENT CONFIGS BEFORE PROCEEDING. I WILL NOT BE RESPONSIBLE IF YOU LOSE YOUR OLD CONFIGS.

Currently, there is no install script. You need to manually move the files to the locations of your existing configs.

```bash
git clone https://github.com/ashyrv0/.dotfiles
cd .dotfiles
```
